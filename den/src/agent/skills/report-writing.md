# Report Writing

## Report Types

### Status Reports
- What's done
- What's in progress
- Blockers
- Next steps

### Analysis Reports
- Executive summary
- Methodology
- Findings
- Recommendations

### Incident Reports
- Timeline
- Impact
- Root cause
- Prevention

## Structure

### 1. Executive Summary
- Key findings in 2-3 sentences
- Main recommendation
- Critical numbers

### 2. Background
- Why this report
- Scope
- Assumptions

### 3. Findings
- Organized by theme
- Data support
- Visualizations

### 4. Recommendations
- Actionable items
- Prioritized
- Resources needed

### 5. Appendix
- Detailed data
- Methodology
- References

## Tips
- Write for busy readers
- Use headers and bullets
- Include visuals
- Be concise
