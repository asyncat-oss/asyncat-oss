# Statistics Interpretation

## Descriptive Stats

### Central Tendency
- Mean: Average (affected by outliers)
- Median: Middle value (robust)
- Mode: Most common

### Spread
- Range: max - min
- Standard deviation: Typical deviation from mean
- Percentiles: P50 = median, P95, P99

### Distribution
- Normal: Bell curve
- Skewed: Asymmetric
- Bimodal: Two peaks

## Common Mistakes

### Correlation ≠ Causation
- A and B correlate
- Doesn't mean A causes B

### Sample Size
- Small samples unreliable
- Check N before trusting

### Selection Bias
- Who is in the sample?
- Does it match your population?

### Survivorship Bias
- Only looking at survivors
- Ignoring dropped outs

## Common Tests

### Comparing Groups
- t-test: Compare means
- ANOVA: Compare 3+ groups

### Relationships
- Correlation: Linear relationship
- Chi-square: Categorical relationships

### Prediction
- Linear regression
- Logistic regression

## Presenting Data

### Visualization
- Bar: Comparisons
- Histogram: Distributions
- Line: Trends
- Scatter: Relationships

### Numbers
- Use P95/P99, not just averages
- Show sample sizes
- Include confidence intervals
