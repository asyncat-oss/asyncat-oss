# Web Research

## Research Process

### 1. Define Scope
- What exactly do you need to know?
- How deep should you go?
- What's the timeframe?

### 2. Source Discovery
- Official documentation
- Academic papers
- Expert blogs
- Community discussions

### 3. Information Gathering
- Note key findings
- Track sources
- Verify claims

### 4. Synthesis
- Organize by theme
- Identify patterns
- Draw conclusions

## Source Types

### Primary
- Official docs
- Source code
- Original research

### Secondary
- Expert analysis
- Tutorials
- Community posts

### Tertiary
- Summaries
- Listicles
- AI-generated

## Quality Indicators
- Author expertise
- Publication date
- Source credibility
- Cross-references

## Documentation
- Always cite sources
- Note the date
- Bookmark key pages
