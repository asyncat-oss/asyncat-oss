# Copywriting

## AIDA Framework

### Attention
- Strong headline
- Start with hook
- Be specific

### Interest
- Build curiosity
- Use details
- Tell a story

### Desire
- Focus on benefits
- Use social proof
- Make it visceral

### Action
- Clear CTA
- Create urgency
- Remove friction

## Powerful Techniques

### Specificity
- "Save 30%" vs "Save money"
- "10,000 users" vs "Many users"

### Clarity
- Short sentences
- Simple words
- One idea per line

### Benefits, Not Features
- "Save 2 hours" not "Fast processor"
- "Stand out" not "Premium design"

## Common Mistakes
- Jargon that alienates
- Weak headlines
- No clear CTA
- Too many options

## Testing
- A/B test headlines
- Test different angles
- Track conversions
