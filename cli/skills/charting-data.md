# Charting Data

## Chart Types

### Comparison
- Bar chart: Categories
- Grouped bars: Multiple categories
- Stacked bars: Total + composition

### Trends
- Line chart: Over time
- Area chart: Cumulative

### Distribution
- Histogram: Frequency
- Box plot: Quartiles
- Scatter: Two variables

### Proportion
- Pie chart: Parts of whole (avoid)
- Donut chart: Slightly better
- Treemap: Hierarchical

## Design Rules

### 1. Show Data, Not Decoration
- No 3D effects
- No unnecessary gridlines
- Minimal colors

### 2. Label Clearly
- Axis labels
- Legend if needed
- Direct labeling when possible

### 3. Appropriate Scale
- Zero baseline for bar charts
- Start at zero not required for line
- Consistent intervals

### 4. Color with Purpose
- Encode data, not decorate
- Consider colorblind users
- Use brand colors intentionally

## Common Mistakes
- Pie charts with many slices
- 3D charts
- Truncated axes
- Too many series
- Missing labels
