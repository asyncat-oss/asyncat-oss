# Inbox Zero

## The Method

### Process Each Email (Once)
- Delete/Archive: Don't need it
- Delegate: Someone else can handle
- Respond: Under 2 min, do it now
- Defer: Schedule time to handle

### Two Key Actions
1. Archive after processing
2. Don't use inbox as to-do

## Quick Decisions

### Delete/Archive
- Old newsletters
- Notifications
- FYIs

### Delegate Forward
- Not your responsibility
- Someone else better suited
- Add context before forwarding

### Respond
- Quick questions
- Short updates
- Under 2 min

### Defer for Later
- Needs more time
- Add to calendar
- Add to task list

## Best Practices

### Don't Live in Email
- Check 2-3 times daily
- Not continuous
- Batch processing

### Use Filters
- Auto-archive certain senders
- Label and filter
- Unsubscribe

### Unsubscribe ruthlessly
- Newsletters = clutter
- One unsubscribe per day
- Quick to leave

### Templates
- Canned responses
- Email templates
- Save time

## Daily Habit
```
Morning: Quick triage (5 min)
Afternoon: Process (15 min)
Evening: Archive/triage (5 min)
```

## Tools
- Archive everything
- Labels for categories
- Search instead of scroll
- Unsubscribe aggressive
