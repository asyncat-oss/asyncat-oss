# Time Blocking

## What is Time Blocking?
- Scheduling specific work into your calendar
- Protecting focus time from meetings
- Batching similar tasks

## How to Time Block

### 1. Know Your Energy
- Morning: High-focus work
- Afternoon: Meetings/admin
- Match tasks to energy

### 2. Identify Priorities
- 2-3 big tasks per day
- Everything else is optional

### 3. Schedule Blocks
- 60-90 min focus blocks
- 25 min for shallow work
- Buffer between blocks

### 4. Protect the Time
- Say no to meetings
- Turn off notifications
- Close email/Slack

## Block Types

### Deep Work
- 60-90 min uninterrupted
- Complex problem solving
- Creative work

### Shallow Work
- 15-30 min
- Email, messages
- Quick decisions

### Admin
- Catch-up time
- Planning
- Reviews

## Best Practices

### Batch Similar Tasks
- All email at once
- All meetings together
- Batch content creation

### Include Buffer
- 5-10 min between blocks
- For transition and notes

### Be Realistic
- Track actual time first
- Adjust expectations

## Template
```
9:00-11:00  Deep Work: [Main task]
11:00-11:15  Break
11:15-12:00  Shallow: [Email/Messages]
12:00-1:00   Lunch
1:00-2:30   Deep Work: [Next task]
2:30-3:00   Meetings
3:00-4:00   Admin/Planning
```
