# Content Writing

## Writing Process

### 1. Research
- Know the audience
- Understand the goal
- Gather key points

### 2. Structure
- Clear headline
- Logical flow
- Strong conclusion

### 3. Draft
- Write first, edit later
- Get the ideas down
- Don't perfect yet

### 4. Refine
- Cut unnecessary words
- Verify claims
- Check flow

## Content Types

### Blog Posts
- Hook in first paragraph
- Subheadings for scanability
- Include examples
- Clear call to action

### Website Copy
- Benefit-focused
- Concise
- Clear navigation

### Marketing
- Lead with value
- Use social proof
- Specific calls to action

## Tips
- Write like you speak
- Use active voice
- One idea per paragraph
- Read aloud to check flow
