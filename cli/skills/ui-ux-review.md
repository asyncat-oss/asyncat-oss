# UI/UX Review

## Evaluation Framework

### Visual Hierarchy
- Most important elements stand out?
- Clear visual flow?
- Proper spacing and alignment?

### Usability
- Easy to understand?
- Intuitive interactions?
- Clear feedback on actions?

### Accessibility
- Color contrast sufficient?
- Keyboard navigable?
- Screen reader friendly?

### Consistency
- Follows platform conventions?
- Consistent spacing/typography?
- Similar patterns used consistently?

## Common Issues

### Navigation
- Where am I?
- How do I go back?
- What can I click?

### Content
- Scannable?
- Too much text?
- Clear headlines?

### Actions
- Primary action obvious?
- Secondary actions de-emphasized?
- Destructive actions clear?

## Feedback Structure
1. What's working
2. Issues (with severity)
3. Recommendations
4. Priority fixes
