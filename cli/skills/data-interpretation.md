# Data Interpretation

## Process

### 1. Understand the Data
- What does each column mean?
- What are the units?
- What's the time range?

### 2. Check Quality
- Missing values?
- Outliers?
- Consistent formatting?

### 3. Explore Patterns
- Distributions
- Correlations
- Trends over time

### 4. Generate Insights
- What does the data tell us?
- What's surprising?
- What should we do?

## Key Metrics

### Descriptive
- Mean, median, mode
- Standard deviation
- Percentiles

### Comparison
- Growth rates
- Conversion rates
- Benchmarks

### Correlation
- Relationships between variables
- Predictive power

## Common Pitfalls
- Correlation ≠ causation
- Ignoring sample size
-Selection bias
- Survivorship bias

## Presentation
- Lead with the insight
- Use visuals wisely
- Be honest about limitations
