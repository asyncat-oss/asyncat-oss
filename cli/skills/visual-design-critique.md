# Visual Design Critique

## Core Principles

### Balance
- Visual weight distribution
- Symmetry vs asymmetry
- White space usage

### Typography
- Readable at all sizes
- Hierarchy clear?
- Pairings harmonious?

### Color
- Contrast accessible?
- Color meaning consistent?
- Palette balanced?

### Imagery
- Photos high quality?
- Icons consistent?
- Illustrations on-brand?

## Critique Steps

### 1. First Impression
- What's the mood?
- What's the main subject?
- What draws the eye?

### 2. Technical Check
- Resolution and scaling
- File formats
- Responsive behavior

### 3. Branding
- Brand alignment?
- Voice consistent?
- Distinctive?

## Deliverables
- Issue list with severity
- Specific recommendations
