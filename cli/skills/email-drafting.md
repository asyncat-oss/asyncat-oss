# Email Drafting

## Email Types

### Informational
- Purpose: Share information
- Clear subject line
- Organized structure

### Request
- Purpose: Get something
- Be specific
- State deadline

### Persuasive
- Purpose: Convince reader
- Lead with benefit
- Clear CTA

### Follow-up
- Purpose: Keep momentum
- Reference previous
- Update status

## Structure

### Subject Line
- Specific and clear
- Action-oriented
- Under 50 chars

### Opening
- Personalized greeting
- Recall shared context
- State purpose

### Body
- 3-5 short paragraphs
- Most important first
- Clear next step

### Closing
- State expected action
- Set deadline if applicable
- Thank reader

## Tone Guidelines

### Formal
- Full words, no slang
- Proper titles
- "Please" and "Thank you"

### Friendly
- Conversational
- Some context
- "Cheers" or "Best"

### Direct
- Get to the point
- Skip pleasantries
- Bullet points ok

## Common Mistakes
- No clear purpose
- Too long
- Missing call-to-action
- Vague subject

## Templates

### Meeting Request
```
Subject: 30-min call to discuss [Topic]

Hi [Name],

[brief context - 1 sentence]

I'd like to [specific goal]. Are you available [timeframe]?

[2-3 time options]

Thanks,
[Your name]
```

### Follow-up
```
Subject: Following up on [Topic]

Hi [Name],

[brief: reference previous interaction]

[Current update - 1-2 sentences]

[Specific ask or next step]

Best,
[Your name]
```
